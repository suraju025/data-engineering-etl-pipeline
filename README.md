# Data Engineering ETL Pipeline

This project demonstrates an end-to-end data engineering pipeline that ingests raw data, cleans and transforms it, and loads analytics-ready tables.

## Tech Stack
- Python
- Pandas
- SQL
- SQLite
- Git

## Workflow
1. Extract CSV & JSON data
2. Clean and transform datasets
3. Create analytics metrics
4. Load data into database
5. Run SQL queries

## How to Run
```bash
pip install -r requirements.txt
python src/main.py
